# Wine Quality Prediction using Machine Learning

## Objective
The objective of this project is to predict the quality of wine based on its chemical properties using different machine learning classification algorithms.

## Dataset
The project uses a wine quality dataset containing chemical features such as acidity, density, alcohol content, pH level, and other characteristics.

## Steps Performed
- Loaded and explored the wine quality dataset
- Performed data cleaning and preprocessing
- Conducted exploratory data analysis using visualizations
- Analyzed feature relationships using a correlation heatmap
- Split the dataset into training and testing sets
- Applied Random Forest, SGD Classifier, and Support Vector Classifier (SVC)
- Compared model performance using accuracy scores
- Evaluated models using confusion matrices and classification reports
- Identified the most important features affecting wine quality

## Tools and Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- VS Code

## Outcome
The project successfully predicted wine quality by analyzing chemical characteristics and comparing multiple machine learning models to identify the best performing algorithm.
